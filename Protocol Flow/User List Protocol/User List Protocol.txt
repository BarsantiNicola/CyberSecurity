title User List Protocol

client->server: USER_LIST
server->client: USER_LIST / ERROR

[message format]

USER_LIST_REQ
{
	Message_Type: 1 byte
	Timestamp: 8 byte
}CIFRED[AES session key]

USER_LIST
{
	Message_Type: 1 byte
	User_List: 0-16384 byte
}CIFRED[AES session key]

ERROR
{
	Message_Type: 1 byte
	Error_type: 1 byte
}CIFRED[AES session key]
